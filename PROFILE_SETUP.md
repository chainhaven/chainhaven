# Chain Haven Profile Setup

## GitHub Profile Repo

Create a new GitHub repository with this exact name:

```text
chainhaven
```

Make it public, then upload the `README.md` from this folder. GitHub will show it on the profile page at:

```text
https://github.com/chainhaven
```

## Local Git Identity

Run these commands on your machine:

```bash
git config --global user.name "Chain Haven"
git config --global user.email "chainhavenlabs@gmail.com"
```

Check the result:

```bash
git config --global user.name
git config --global user.email
```

## First 30 Days

Week 1:

- Learn GitHub basics
- Set up profile README
- Learn Solidity fundamentals
- Build a simple smart contract

Week 2:

- Learn Foundry or Hardhat
- Write tests for smart contracts
- Deploy to a testnet
- Document everything in GitHub

Week 3:

- Learn wallet connection with React
- Build a simple dApp frontend
- Read contract data from a testnet
- Push clean commits daily

Week 4:

- Build one polished mini project
- Write a clear project README
- Add screenshots
- Share your work publicly
